{{'How to' | translate}}

Commencer par ajouter une nouvelle question en cliquant sur "Ajouter une nouvelle question". Une boîte rouge apparaîtra. Elle est rouge car votre question n'est pas publiée (même si elle est enregistrée en base de données). Vous publierez votre question que lorsqu'elle sera entièrement rédigée

Toutes les actions que vous faîtes sont sauvegardées en direct (il n'y a pas de bouton valider). Faites donc attention ! par exemple, supprimer une question est irréversible

Dans la case "Votre question", entrez votre question

Obligez-vous à ajouter des tags (ceci pour faciliter les recherches ultérieurement)

Ajouter autant de choix que vous désirez en cliquant sur "Nouveau choix"

Un choix sera correct si le numéro dans la bulle rouge ou verte est positif (strictement). Si ce numéro est nul ou inférieur à zéro, c'est une mauvaise réponse

Dans le dernier cadre ajoutez l'explication. Si vous voulez de l'italique ou du gras, utilisez le langage markdown. Pour faire un renvoi ver un point de règle, utilisez la notation suivante (ici, renvoi vers le point 9.3.3 des règles 2014): [2014-9.3.3](/#rules2014/9.3.3)
